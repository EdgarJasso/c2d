Chat2desk

Que es Chat2desk OnCloud

n una plataforma que te permitirá una efectiva comunicación Business-to-Client. Ya que es centro unificado de mensajerías instantáneas que permitirá atender a tus clientes por medio de diferentes canales de mensajería instantánea

Reportes Estadisticos

Los reportes estadísticas son un conjunto de datos con mas valores para analizar el rendimiento de su centro unificado de mensajerías instantáneas, como tiempo de procesamiento y reacción, cantidad de mensajes, clientes y solicitudes, acciones de los operadores, Puntuaciones recibidas y más.

  • Las estadísticas se calculan una vez al día a la 1 am, hora local. El cálculo está desactivado por defecto
  • Solicite al administrador que lo active para su empresa.
  • La mayoría de los valores se calculan en base a la solicitud/requisición

Solicitud/requisición - es un conjunto de mensajes dentro de un diálogo con un cliente. Como regla general, la solicitud comienza con el primer mensaje del cliente y finaliza cuando el diálogo está cerrado. En caso de que el cliente vuelva a escribir cuando el dialogo ya se cerro, el sistema iniciara una nueva solicitud.

  • Para activar la visualización de solicitudes en sus chats, se puede hacer en Ajustes & Configuración / Etiquetas y Solicitudes, solo el administración puede habilitar esa funcionalidad.
  • Solicite al administrador que lo active para su empresa.
  • El período de tiempo siempre se calcula en segundos.

Filtros en Chats

Puedes crear conjunto de filtros personales para tus Chats, en el menú de a izquierda para un rápido acceso. Por ejemplo, puedes crear estos filtros de chats:

  • Chats vencidos por operador o por grupos de operadores
  • Chats por tipo de mensajería por país
  • Chats con ciertas etiquetas (tags)
  • Chat con ciertos clientes VIP

Cuando un conjunto es seleccionado, de inmediato los chats que cumplen con ese criterio son seleccionados. Si el conjunto esta vacío significa que no tiene chats. Dicha funcionalidad se torna semitransparente para evitar distractores. El número de chats en cada conjunto se recalcula automáticamente cuando se actualízala página o cuando selecciona un conjunto. Así tendrás la capacidad de checar chats interesantes de forma rápida.

Plantillas Predefinidas

Crea de forma sencilla plantillas con la finalidad de acelerar la respuesta por parte de los agentes. Estas pueden ser creadas de forma personalizable hacia el cliente, ya sea con texto, imágenes o PDF, de igual forma dichas plantillas pueden ser categorizadas para una más fácil administración.

Plantillas Predefinidas

Las etiquetas (Tags) son ampliamente utilizadas en el sistema. Además de las marcas visuales para los operadores, permiten filtrar a los clientes, ver estadísticas sobre solicitudes de diferentes temas, restringir el envío de ofertas para calificar para chatear, configurar cadenas de mensajes y mucho más.

Además, es posible ejecutar un programa o script específico cuando se aplica o elimina una etiqueta de cliente. Puede realizar operaciones masivas con etiquetas utilizando mensajes masivos y operaciones masivas. Las etiquetas se pueden aplicar tanto a los clientes como a sus solicitudes.

Dentro de esta opción Clientes y solicitudes puedes ser etiquetados. Una solicitud es una parte del dialogo de chat con el cliente la cual puede contener mucho intercambio de mensajes, usualmente la solicitud es iniciada por el cliente y termina cuando el chat es cerrado. Si la comunicación continua después de que el dialogo es cerrado entonces se iniciara una nueva solicitud en el mismo dialogo.

Las etiquetas y grupo de etiquetas son administradas en el menú de “Ajustes & Configuración”

Menú de Autoservicio

Es un set de comandos con el cliente interactúa como si estuviera en un IVR pero en chat. Esta funcionalidad ayuda al operador a tener una mejor percepción de lo que el cliente quiere. Hay ciertos canales de mensajería instantánea que soportan el manejo de botones como Facebook Messenger, Viber Public, Telegram y el Chat en Línea. El menú es visualizado en forma de botones

Menú de Autoservicio

El bot de chat responde automáticamente a los clientes en lugar de a los operadores, asigna etiquetas o asigna el chat a un operador específico o a un grupo de atención especializada. ChatB0ot analiza todos los mensajes entrantes y actúa de acuerdo con las palabras clave identificadas.

Crea un set y dentro de cada set puede haber al menos un grupo de palabras clave, puedes crear tantos grupos como tu set lo requiera. Una respuesta especifica es contestada al cliente solo si en el mensaje del cliente contiene al menos una palabra de cada grupo y no contiene ninguna palabra de paro.

Puedes crear los sets de preguntas que operación requiera, no hay una limitante tecnológica

Envíos & Operaciones Masivas

Chat2desk permite el envío de mensajes a tus clientes actuales. Puedes seleccionar a los clientes con base a las siguientes reglas

  • Etiquetas y mensajeros: los mensajes se enviarán a los clientes, cuyas tarjetas de información contienen etiquetas y mensajeros específicos.
  • Filtro del diálogo con base a fecha de inicio y fecha de término. Los mensajes se enviarán a los clientes, cuyos diálogos ocurrieron dentro de las fechas especificadas.
  • Cantidad límite: establezca el número de destinatarios. Escriba 0 para el número ilimitado de destinatarios.
  • Contabilización: esta sección permite calcular previamente el número de destinatarios.
  • Texto e imagen / PDF: especifique un texto, una imagen o un documento PDF para mensajes masivos

Módulo de Túneles & Bots

Los túneles le permiten enviar mensajes a los clientes, respondiendo a sus preguntas y realizar diversas acciones dependiendo de la actividad o inactividad del cliente. Los túneles son activados por varios eventos, por ejemplo, cuando se accede a un nuevo cliente o cuando se le asigna una etiqueta.

Alcance de los túneles.

  • Creando cuestionarios. Puede interrogar a sus clientes, asignar etiquetas en función de sus respuestas o guardar respuestas en campos adicionales. De esta manera, puede segmentar a sus clientes y aplicar un enfoque diferente a cada uno de ellos
  • Mensajes de calentamiento. Envíe varios mensajes al cliente con información útil con un retraso para “calentar” al cliente; Finalmente envíe un enlace a su producto
  • Soporte técnico. Haga preguntas importantes para resolver problemas técnicos.
  • Cobros por PayPal o Mercado pago

WhatsApp Business

WhatsApp Business es una aplicación de descarga gratuita disponible para Android y iPhone, que ha sido desarrollada especialmente para pequeñas y medianas empresas. WhatsApp Business facilita las interacciones con los clientes, ya que ofrece herramientas para automatizar, organizar y responder rápidamente a los mensajes. Está diseñada y funciona de forma similar a WhatsApp Messenger. Puedes usarla para seguir haciendo todas las cosas que ya haces como enviar mensajes y fotos.

Algunas de las funciones que tiene esta aplicación son:

  • Perfil de empresa para mostrar la información más importante, como tu dirección, correo electrónico y sitio web.
  • Estadísticas para ver cuántos mensajes se enviaron, se entregaron y se leyeron exitosamente.
  • Herramientas de mensajería para responder de forma rápida a tus clientes.

WhatsApp Business

WhatsApp ha reforzado las reglas de bloqueo para envíos masivos y boletines.
Lea las nuevas recomendaciones para usar opciones como «Contactar Cliente Nuevo» «Túneles de ventas».

  1. La función «Contactar Cliente Nuevo» ya no está disponible durante el período de trial.
  2. Ahora WhatsApp bloquea para 5 o más mensajes idénticos a nuevos clientes. Por lo tanto, es necesario comprender que, si envía mensajes idénticos a nuevos clientes en la versión de pago mensual del servicio, el número se bloqueará de inmediato. Nuestro servicio también se opone a los boletines de noticias para los nuevos clientes.
  3. Al enviar a la base de clientes actual, también existe el riesgo de ser bloqueado si envía más de 20-30 mensajes idénticos al día.
  4. Los riesgos se aplican a los túneles de ventas si envía más de 20 a 30 mensajes idénticos por día a través de la base actual de clientes.
  5. Si la cuenta está muy activa, la cantidad de mensajes enviados válidos puede aumentar. Sin embargo, esto no elimina completamente el riesgo de ser bloqueado cuando se usa la función "Escribir primero" y "Túneles de ventas".

Mantenga estas recomendaciones en mente e intente encontrar formas de alentar al cliente a que le escriba primero.
